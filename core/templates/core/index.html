🛍️ Online Store - Backend API

Capstone project showcasing backend features for an eCommerce platform.

📦 Product Management

  • /store/products/ – List & Create Products
  • /store/products/<id>/ – View, Update, Delete Product
  • /store/products/<product_id>/reviews/ – Product Reviews
  • /store/products/<product_id>/images/ – Product Images

📁 Collection Management

  • /store/collections/ – List & Create Collections
  • /store/collections/<id>/ – View a Collection

🛒 Cart Management

  • /store/carts/ – Create a Cart
  • /store/carts/<cart_id>/items/ – Add/View Items

👤 Customer Management

  • /store/customers/ – View & Edit Customer

🧾 Order Management

  • /store/orders/ – Place/View Orders

🔐 Authentication

  • /auth/users/ – Register
  • /auth/jwt/create/ – Login
  • /auth/jwt/refresh/ – Refresh Token
